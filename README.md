## Contributing

Contributions are welcome! Please follow these simple steps:

1. Fork the repository.

2. Create a new branch for your sample code and explanation.

3. Add your code sample or note in the appropriate section (keep it clear and concise).

4. Ensure the code is tested or verified in a real Frappe environment.

5. Submit a pull request with a short description of what you added.

If you're unsure about something, feel free to open an issue before contributing.
